The fastest way to get Bolly running. A single curl command downloads the latest binary, creates your config, and sets up a background service.

## Install

```bash
curl -fsSL https://bollyai.dev/install.sh | bash
```

This script will:

1. Detect your operating system and architecture
2. Download the latest Bolly binary from GitHub releases
3. Place it in a system-accessible location
4. Create the default config directory at `~/.bolly/`
5. Generate a default `config.toml`
6. Set up a background service (systemd on Linux, launchd on macOS)

<Info>The install script is open source. You can [review it on GitHub](https://github.com/triangle-int/bolly) before running.</Info>

## Open Bolly

The install script starts the service automatically. Open [http://localhost:26559](http://localhost:26559) in your browser — Bolly will guide you through the initial setup.

During onboarding you'll choose how to connect to Claude:

- **API Key** — bring your own key from [console.anthropic.com](https://console.anthropic.com). Pay per usage.
- **Claude Pro / Max subscription** — use your existing subscription instead of an API key.

<Info>See the [API Keys guide](/installation/api-keys) for details on both options.</Info>

## Managing the service

### Check status

<CodeGroup>

```bash Linux
sudo systemctl status bolly
```

```bash macOS
launchctl list | grep bolly
```

</CodeGroup>

### Stop the service

<CodeGroup>

```bash Linux
sudo systemctl stop bolly
```

```bash macOS
launchctl unload ~/Library/LaunchAgents/dev.bollyai.bolly.plist
```

</CodeGroup>

### Restart

<CodeGroup>

```bash Linux
sudo systemctl restart bolly
```

```bash macOS
launchctl unload ~/Library/LaunchAgents/dev.bollyai.bolly.plist
launchctl load ~/Library/LaunchAgents/dev.bollyai.bolly.plist
```

</CodeGroup>

### View logs

<CodeGroup>

```bash Linux
sudo journalctl -u bolly -f
```

```bash macOS
tail -f ~/.bolly/bolly.log
```

</CodeGroup>

## Updating

To update Bolly to the latest version, run the install script again:

```bash
curl -fsSL https://bollyai.dev/install.sh | bash
```

The script detects existing installations and performs an in-place upgrade while preserving your config and data. See the [full updating guide](/installation/updating) for the built-in update command and release channels.

## Uninstalling

Run the uninstall script to remove Bolly:

```bash
curl -fsSL https://bollyai.dev/uninstall.sh | bash
```

This stops the service, removes the binary and service files, and cleans up your PATH. See the [full uninstall guide](/installation/uninstall) for options like keeping your data.
