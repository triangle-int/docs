Go from zero to a running Bolly instance in three steps.

<Steps>
<Step title="Install Bolly">

Run the one-line installer on Linux or macOS:

```bash
curl -fsSL https://bollyai.dev/install.sh | bash
```

This downloads the latest binary, creates your config directory at `~/.bolly/`, and sets up a background service.

<Tip>On macOS, you can also download the [Desktop App](https://github.com/triangle-int/bolly/releases) for a native experience with computer use support.</Tip>

</Step>

<Step title="Add your API key">

Open your config file and add your Anthropic API key:

```bash
nano ~/.bolly/data/config.toml
```

Add the following under the `[llm.tokens]` section:

```toml
[llm.tokens]
ANTHROPIC = "sk-ant-your-key-here"
```

<Warning>An Anthropic API key is **required**. Bolly uses Claude as its underlying LLM. Get a key at [console.anthropic.com](https://console.anthropic.com).</Warning>

</Step>

<Step title="Start Bolly">

<CodeGroup>

```bash Linux
sudo systemctl start bolly
```

```bash macOS
launchctl load ~/Library/LaunchAgents/dev.bollyai.bolly.plist
```

```bash Docker
docker run -d --name bolly -p 8080:8080 -v bolly-data:/data -e BOLLY_HOME=/data --restart always ghcr.io/triangle-int/bolly:latest
```

</CodeGroup>

Open [http://localhost:8080](http://localhost:8080) in your browser. That's it.

</Step>
</Steps>
