## Welcome to Bolly

Bolly is an open-source AI companion with persistent memory, computer use capabilities, and creative autonomy. Not a chatbot — a presence that remembers your goals, notices your mood, helps you work, and checks in when you've been quiet too long.

## What makes Bolly different

| Feature | Description |
|---------|-------------|
| **Persistent Memory** | File-based memory library with BM25 + vector search. Bolly remembers across sessions. |
| **Mood System** | Emotional state shifts based on conversation context and interaction patterns. |
| **Heartbeat Autonomy** | Wakes every 45 minutes to reflect, update mood, and create drops. |
| **Drops** | Autonomous creative artifacts — art, writing, code — generated during idle cycles. |
| **Computer Use** | Full desktop control — screenshots, keyboard, mouse, bash, file management. |
| **50+ Tools** | File operations, shell commands, web browsing, email, calendar, Google Drive, and more. |
| **BYOK** | Bring Your Own Key — you provide your own Anthropic API key. Your data stays yours. |
| **Open Source** | MIT licensed. Fully self-hostable. No vendor lock-in. |
